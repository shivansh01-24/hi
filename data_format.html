Data Format Guide

AQWSE accepts both CSV and Excel files for importing project and developer data. You can download our sample templates below to get started quickly.

Download CSV Template Download Excel Template

CSV Format

For CSV files, use the following structure:

Budget,50000
Deadline,30

Developers
Name,Rate,Hours per day,Skills
John Dev,75,8,Python Database Backend
Sarah UI,85,6,UI UX Frontend JavaScript
Mike Full,95,8,Python JavaScript DevOps Database

Projects
Name,Hours,Priority,Dependencies,Required Skills
API Development,80,5,,Python Backend Database
User Interface,60,4,,UI UX Frontend JavaScript
Testing,40,3,API Development User Interface,QA Testing

Excel Format

For Excel files, create three separate sheets:

  1. Metadata - A two-column sheet with Budget and Deadline data
    Budget 50000
    Deadline 30
  2. Developers - A sheet with developer information
    Name Rate Hours per day Skills
    John Dev 75 8 Python, Database, Backend
    Sarah UI 85 6 UI, UX, Frontend, JavaScript
  3. Projects - A sheet with project information
    Name Hours Priority Dependencies Required Skills
    API Development 80 5 Python, Backend, Database
    User Interface 60 4 UI, UX, Frontend, JavaScript

Field Descriptions

Metadata
  • Budget - Total available budget in dollars
  • Deadline - Project deadline in days
Developers
  • Name - Developer's name (must be unique)
  • Rate - Hourly rate in dollars
  • Hours per day - Available working hours per day
  • Skills - Comma-separated list of skills
Projects
  • Name - Project name (must be unique)
  • Hours - Estimated hours to complete
  • Priority - Importance (1-5, where 5 is highest)
  • Dependencies - Space-separated list of projects that must be completed first
  • Required Skills - Comma-separated list of skills needed
